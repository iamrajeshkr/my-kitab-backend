Bingent API

AI control plane — guest auth, RAG, recommendations, companion. This is an API; call it over HTTP. ● = public.

GET /health ● — liveness
POST /v1/auth/guest ● — create guest user, mint token
POST /v1/compose-page — tonight's page (RAG + sources)
POST /v1/ask-line — answer about a highlighted line
POST /v1/companion — Ask Bingent (reply + recommendations)
POST /v1/recommend — CF + weather-fit picks
POST /v1/search — feeling-based discovery
POST /v1/reflections — store + enrich a reflection
POST /v1/weather — inner-weather check-in
POST /v1/events — batched event ingest
GET /v1/garden — felt-progress summary
· /v1/mirror · /v1/letter · /v1/sit · /v1/practices — portrait, letter, daily sit, practices

All /v1/* except /v1/auth/guest require Authorization: Bearer <token>.
